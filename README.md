#    「 ADIIXYZBOT 」
Simple WhatsApp Bot

# INSTALL

## FOR TERMUX/VPS USERS
```cmd
apt update && apt upgrade
apt install git
apt install nodejs
apt install ffmpeg
apt install imagemagick
git clone https://github.com/Adiixyz/simple-botwa
cd simple-botwa
npm install
```

###### Run
```cmd
node adiixyz.js
```
---------
